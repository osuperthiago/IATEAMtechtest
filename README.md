## Readme referende à pasta 3-automation

## Preparação de ambiente da automação:

1 - Instalar:  **Node.js**

1 - Baixar e instalar a versão LTS adequada para seu sistema operacional:
2 - [https://nodejs.org/pt-br/download](https://nodejs.org/pt-br/download)
3 - Seguir os botões de continuar ate finalizar a instalação.
4 - Reiniciar o PC.
( OBS: caso o node pare numa tela azul, fechar a tela e reiniciar o pc)

2 - Baixar:  **Pasta do GIT**

1 - Baixar a pa pasta do projeto no Github:
2 - Abrir: https://github.com/osuperthiago/IATEAMtechtest
3 - Clicar em "code".
4 - Clicar em Download ZIP;
5 - Descompactar a pasta em alguma pasta local.

## Para rodar direto do CMD

1 - Abrir o command: 
2 - Atalho: windows + R, digitar cmd e apertar enter.
3 - Usar o comando cd para mudar para a pasta 3-automation que foi decompactada anteriormente.
4 - Exemplo: cd C:\GitRepo\10-IATECAM\3-automation
5 - Rodar o comando: npx cypress run
6 - Acompanhar a automação e verificar o relatório no final.

## Para rodar por um IDE

1 - Abrir um IDE (Sugestão Visual Scudio Code).
( Caso não tenha O VSC seguir os passos no final desse readme e volte).
2 - Clicar em File e Open Folder.
3 - Procurar a pasta 3-automation, dois cliques nela.
4 - Abrir um novo terminal no submenu da barra superior New Terminal.
5 - Rodar o comando: npx cypress open
6 - Clicar em E2E Testing
7 - Clicar em um um navegador e depois em Start E2E Testing.
8 - Clicar em Cenário1.cy.js
9 - Observar a automação em curso.

OBS: Instalação do VSC:
Baixar e instalar a versão adequada para seu sistema operacional:
[https://code.visualstudio.com/download](https://code.visualstudio.com/download)
Seguindo os botões de continuar ate finalizar a instalação.
Reiniciar o pc.
